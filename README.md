# Portfolio

A modern personal portfolio built with **Next.js** and **Tailwind CSS**.

## ✨ Features

- Responsive layout for desktop and mobile
- Smooth scrolling and custom CSS animations
- Reusable utility classes for gradients, hover effects, and motion
- Clean, component-based project structure

## 🛠 Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- TypeScript (if enabled in your project)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm (or pnpm/yarn)

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```

Open: `http://localhost:3000`

## 📦 Build for production

```bash
npm run build
npm start
```

## 📁 Project Structure (typical)

- `app/` — App Router pages/layouts
- `app/globals.css` — global styles, custom keyframes, utility classes
- `public/` — static assets
- `components/` — reusable UI components

## 🎨 Styling Notes

Global animation utilities are defined in:

- `app/globals.css`

Examples include:

- `.animate-fade-in`
- `.animate-slide-in-left`
- `.animate-slide-in-right`
- `.animate-float`
- `.hover-lift`
- `.gradient-text`
